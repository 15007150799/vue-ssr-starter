# vue-ssr-starter

> A Vue.js starter kit with vue 2.0, vue-router, vuex and axios for server-side rendering.

![image](https://user-images.githubusercontent.com/5000396/27516172-0c2a6828-59e7-11e7-8d7c-df32d538ff3a.png)

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```

# vue-ssr-starter

> A Vue.js starter kit with vue 2.0, vue-router, vuex and axios for server-side rendering.

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```

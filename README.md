# vue-ssr-starter

> A Vue.js starter kit with vue 2.0, vue-router, vuex and axios for server-side rendering.

![screenshot](https://user-images.githubusercontent.com/5000396/27325686-c9ccec44-55db-11e7-8c05-a3d8a4cd0415.png)

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```

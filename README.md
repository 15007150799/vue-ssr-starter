# vue-ssr-starter

> A Vue.js starter kit with vue 2.0, vue-router, vuex and axios for server-side rendering.

![screenshot](https://user-images.githubusercontent.com/5000396/27325896-6e9383b4-55dc-11e7-8010-3ea3230e8cbb.png)

## Build Setup

``` bash
npm install
npm run build
npm start
```

## Development Setup

```bash
npm install
npm run dev
```
